# Минимальные пути к зданиям: v5 vs fix4

**Методология:** жадный по `total_cost` — на каждом шаге берём науку с наименьшим `total_cost` среди тех, что дают уровни нужного здания, пока не наберём целевое количество уровней. `total_cost` = собственная стоимость науки + все предки по DAG без задвоений.
